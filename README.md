# 新正方教务查询

#### 介绍
1. 本系统使用SSM框架进行开发。
2. 目前完成“本学期成绩查询”、“按学期成绩查询”，以及“课表查询”的功能部分，“课表查询”的前端还没有做。
3. 要求教务系统可以公网访问，或者配合学校vpn使用。

#### 软件架构
软件架构说明


#### 安装教程

1. 导入idea
2. 部署tomcat
3. 访问localhost:8080/jiaowu/index

#### 获取信息流程

1. 获取csrftoken和Cookies
2. 请求获取PublicKey
3. 利用PublicKey对登录密码加密并用Base64编码
4. 进行登录
5. 获取所需要的信息
